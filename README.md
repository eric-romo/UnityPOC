##Hotkeys
- Left Alt	              > Superkey
- SK + N									> New Window
- SK + W									> Close Window
- SK + [Arrow Key]        > Move display to new location
- SK + ,	                > Go Back
- SK + .	                > Go Forward 
- SK + [1-5]		          > Go To Favorite
- [Shift] + Drag Display	> Drag display along axis
